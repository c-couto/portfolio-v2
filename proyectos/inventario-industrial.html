CASE STUDY 01

Sistema Inteligente de Consulta de Activos Industriales

Python · Ubuntu LTS · VMware · HTML · JSON · Linux

Problema

Las consultas al inventario requerían abrir múltiples planillas Excel, dificultando el acceso rápido a la información.

Solución

Desarrollé una aplicación web en Python publicada sobre Ubuntu Server LTS (VMware) accesible desde la intranet. El sistema aplica reglas de negocio y consulta una base de conocimiento compuesta por archivos JSON y TXT.
